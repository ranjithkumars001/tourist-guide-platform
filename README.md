# 🌍 Tourist Guide Platform

A full-stack MERN Travel Planning application that helps users discover destinations, create travel itineraries, and manage trips.

## 🚀 Features

- User Authentication (JWT)
- Login & Registration
- Protected Dashboard
- Explore Destinations
- Destination Details
- Trip Planner
- Create Trip
- Edit Trip
- Delete Trip
- MongoDB Atlas Integration
- Responsive UI

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router DOM

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- Bcrypt.js

## 📂 Project Structure

```
tourist-guide-platform
│
├── client
└── server
```

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/ranjithkumars001/tourist-guide-platform.git
```

Install frontend

```bash
cd client
npm install
npm run dev
```

Install backend

```bash
cd server
npm install
npm run dev
```

## 🌐 API

- POST /api/auth/register
- POST /api/auth/login
- GET /api/auth/me
- GET /api/destinations
- GET /api/trips
- POST /api/trips
- PUT /api/trips/:id
- DELETE /api/trips/:id
